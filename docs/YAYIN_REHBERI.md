# Yayin Rehberi

## GitHub Pages

1. Repository adi `kelime-evreni` olacak sekilde GitHub'da public repo olusturun.
2. Proje dosyalarini `main` dalina yukleyin.
3. Repository ayarlarindan **Pages** bolumune girin.
4. Source olarak `Deploy from a branch` secin.
5. Branch: `main`, folder: `/root` secin.

## Vercel

1. Vercel hesabinizdan yeni proje olusturun.
2. GitHub repository olarak `kelime-evreni` deposunu secin.
3. Framework preset: `Other`
4. Build command bos birakilabilir.
5. Output directory bos veya `/` olarak kalabilir.

## Not

Proje statik HTML/CSS/JavaScript ile hazirlandigi icin ek derleme adimi gerektirmez.
